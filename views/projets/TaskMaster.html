TaskMaster - Application de gestion de tâches



Le contexte du projet


    Ce projet, nommé TaskMaster, est une application de gestion de tâches développée en binôme. L'objectif était de concevoir et de développer une application multiplateforme en utilisant des technologies spécifiques et en adoptant une gestion de projet de type AGILE.



L'ambition du projet

    L'ambition principale de TaskMaster était de fournir un outil complet permettant aux utilisateurs de gérer leurs tâches de manière efficace. L'application devait permettre à un utilisateur de :

  • Créer un compte et s'authentifier.
  • Créer, modifier, et supprimer des tâches.
  • Ajouter des sous-tâches, des étiquettes et des commentaires.
  • Suivre l'évolution d'une tâche grâce à son statut.
  • Associer une tâche à un projet et la confier à une autre personne.
  • Consulter l'historique des modifications.
  • Visualiser les tâches selon leur priorité, leur échéance ou leur catégorie.

Technologies utilisées

Pour ce projet, j'ai utilisé les langages et technologies suivants :

  • **Langage de programmation :** C#
  • **Frameworks :** .NET MAUI pour le développement multiplateforme.
  • **Design Pattern :** Le modèle de conception MVVM (Model-View-ViewModel).
  • **Base de données :** MySQL pour le stockage des tâches.
  • **Gestion des données :** Entity Framework Core pour interagir avec la base de données.
  • **Outils de gestion de projet :** GitHub pour les dépôts de code et la gestion de projet.

Compétences Développées

Techniques :

  • Maîtrise du framework .NET MAUI et du langage C# pour une application multiplateforme.
  • Implémentation du pattern de conception MVVM.
  • Utilisation d'Entity Framework Core pour la gestion des données.
  • Conception et création d'une base de données relationnelle (MySQL).

Transversales :

  • Gestion de projet en binôme selon la méthodologie AGILE (Scrum).
  • Répartition des tâches et suivi de l'avancement.
  • Utilisation de GitHub pour le versionnement et la gestion de projet.

Humaines :

  • Collaboration efficace et communication régulière au sein de l'équipe.
  • Résolution conjointe des problèmes techniques et des défis rencontrés.

Conclusion : Retour d’expérience et apprentissage

Ce projet a été l'occasion de mettre en pratique des compétences en développement avancé et de se familiariser avec l'écosystème .NET MAUI. La conception d'une application multiplateforme et l'adoption du pattern MVVM ont été particulièrement enrichissantes. La gestion de la base de données via Entity Framework Core a permis une bonne compréhension de la persistance des données. Le principal défi a été lié à l'intégration des différentes technologies. Le rendu final est pleinement fonctionnel et démontre une bonne maîtrise des concepts étudiés.
